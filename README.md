Hola

Para usar este programa, necesitas los siguientes paquetes:
-Python-tk
-Pandas
-openpyxl

Además, es necesario crear una carpeta llamada output en el directorio de ejecución.

Se requiere crear carpetas por año de datos, pueden ser descargadas en datos.gob.cl , posteriormente es necesario agregar los datos en estas carpetas anualizadas y luego ejecutar el código. El código funciona con 3 variables:

AÑOS: Carpetas donde hará la búsqueda.

Puedes elegir cual de los 2 tipos de datos buscar: 
ARANC_NAC: Código arancelario del producto buscado.
Número único de importador: Número único de importador (Clave).

Los archivos pueden leerse con el proyecto ImportacionesChile_exe
